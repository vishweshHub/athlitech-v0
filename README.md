# AthliTech

## 🚀 Overview
AthliTech is a mobile-first application designed to help athletes track workouts, fatigue, and recovery to improve performance and consistency.

---

## ❗ Problem
College athletes often struggle with:
- Injury due to overtraining
- Fatigue without proper tracking
- Inconsistent training habits

There is no simple system to monitor daily performance and recovery.

---

## 💡 Solution
AthliTech provides a structured way to:
- Log daily workouts
- Track fatigue levels
- Monitor recovery
- Build consistent training habits

---

## ⚙️ Tech Stack
- Backend: Node.js, Express.js
- Database: SQLite
- Mobile (Planned): React Native

---

## 🧩 Features (v0)
- Workout logging system
- Basic data storage
- Structured input for tracking performance

---

## 🔄 Future Improvements
- Mobile app using React Native
- Performance analytics dashboard
- AI-based training insights

---

## 🎯 Goal
To build a practical tool that helps athletes train smarter, avoid injuries, and stay consistent.
